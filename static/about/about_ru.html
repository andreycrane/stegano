О проекте

Stegano - это демонстрационное HTML5-приложение, реализующее стеганографию в браузере без применения серверной стороны. Стеганография в данном приложении заключается в следующем:

  • Вы загружаете имеющийся у вас файл изображения формата BMP
  • Приложение проверяет файл на наличие скрытой в нем (посредством данного приложения) ранее информации
  • Если таковая была найдена, она будет прочитана и выведена в диалоговом окне
  • Если информация не существовала, её можно ввести в том же диалоговом окне, после чего создается копия загруженного файла, содержащая скрытый текст
  • Сохранив копию себе на компьютер, Вы вновь сможете увидеть скрытое посредством Stegano

Я разработал данное приложение для демонстрации применения Drag and Drop и File API, а также API типизированных массивов клиентского JavaScript, но если Вы найдете ему применение, пожалуйста отблагодарите меня просто кликнув по одной из социальных кнопок расположенных ниже.
